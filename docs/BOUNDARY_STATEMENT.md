# PO10D Boundary Statement

This release is a third-party reproduction package for the Wuxing V32/V33/V34 physical-engine closure chain.

It is allowed to:

- rebuild RGESolver and HEPfit from source,
- rerun V32 white-box and W-mass scheme assertions,
- rerun V33 EWPO forward vector closure,
- rerun V34 Higgs and Flavour C9_22 forward closure,
- rerun PO09R/PO09W/PO09X deterministic reproduction gates.

It is not allowed to:

- read external W-mass targets,
- run fits or MCMC,
- tune physics parameters,
- change physics formulas,
- claim new physics equivalence from this package alone.
